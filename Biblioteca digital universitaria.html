¿Qué es HTML?

HTML es un lenguaje de marcado utilizado para crear y estructurar páginas web. Permite organizar información mediante títulos, párrafos, imágenes, tablas, enlaces y formularios.

Aplicación en este proyecto

En esta biblioteca digital universitaria, HTML permite organizar libros, autores, categorías académicas y recursos de consulta para estudiantes.

Características principales de HTML

  • Organiza el contenido web.
  • Permite insertar imágenes.
  • Facilita la navegación mediante enlaces.
  • Permite crear formularios y tablas.
  • Es la base del desarrollo web.
